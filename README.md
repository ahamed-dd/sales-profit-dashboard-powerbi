## Data & Process
- **Source**: Daily sales data maintained in Excel files  
- **Data Cleaning & Transformation**:
  - Combined daily Excel files using **Power Query**
  - Cleaned inconsistent product, buyer, and route names
  - Standardized date formats and calculated profit
- **Modeling & Visualization**:
  - Product-wise, route-wise, and buyer-wise analysis
  - Time-based profit trends for operational tracking
